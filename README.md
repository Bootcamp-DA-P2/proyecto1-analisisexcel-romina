# proyecto1-analisisexcel-romina
# 📊 Análisis de datos y dashboard interactivo en excel 
🧭 Objetivo del proyecto

Este proyecto tiene como objetivo aplicar un flujo completo de análisis de datos utilizando Microsoft Excel, desde la importación y limpieza de datos hasta la creación de un dashboard interactivo para la exploración de insights.

📦 Dataset utilizado

Se utiliza el dataset Kiva Crowdfunding (Data Science for Good), que contiene información sobre microcréditos otorgados en distintas regiones del mundo.

El análisis se realiza sobre una muestra de 10.000 registros, con el objetivo de optimizar el rendimiento en Excel.

🛠️ Metodología aplicada

🟢 1. Preparación de datos
- Importación de datos en Excel
- Limpieza y transformación básica
- Eliminación de registros no necesarios
- Organización del dataset en formato tabla

🟡 2. Análisis de datos
- Creación de tablas dinámicas para exploración de variables
- Análisis de métricas como:
- distribución de préstamos por país
- análisis por sector económico
- segmentación por género
- Uso de gráficos para visualización de patrones
- 

🟠 3. Dashboard interactivo

Construcción de dashboard en Excel
Visualización de KPIs mediante tablas dinámicas
Uso de gráficos dinámicos para análisis visual

🎯 Conclusión
